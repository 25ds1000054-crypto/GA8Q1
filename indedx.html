Example: Calculating Discounted Cash Flow (DCF)

// Simple DCF: present value of cash flows
function dcf(cashFlows, discountRate) {
  return cashFlows.reduce((pv, cf, i) => pv + cf / Math.pow(1 + discountRate, i + 1), 0);
}

const cashFlows = [100, 110, 120, 130];
console.log(`DCF = $${dcf(cashFlows, 0.08).toFixed(2)}`);
Walk through the code and highlight assumptions: discount rate, timing, and forecast certainty.
